# Frontend (src)

Эта папка содержит исходный код клиентской части приложения, построенный на современных React/JS стеках.

## Архитектура
Используется Feature-Sliced Design (или близкая структура):
- \`app/\` — инициализация приложения, глобальные провайдеры, роутинг.
- \`pages/\` — страницы маршрутизации.
- \`features/\` — бизнес-логика.
- \`shared/\` — переиспользуемые компоненты: UI-кит, утилиты.
- \`stores/\` — глобальное состояние платформы (например, через zustand).

Код взаимодействует с Rust-бэкендом с помощью \`@tauri-apps/api\`.
