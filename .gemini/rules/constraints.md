# КРИТИЧЕСКИЕ ОГРАНИЧЕНИЯ (Красные флаги)
- macOS only (darwin-arm64). Телефид — чисто нативное Tauri-приложение с жесткой привязкой к TDLib. Никаких браузерных моков или адаптеров для запуска/теста фронтенда вне Tauri среды создавать НЕЛЬЗЯ.
- `feedStore` строго НЕ зависит от `uiStore`.
- Батчинг `feed_updated` в Rust 500ms loop, не спамить emit'ами на фронт.
- Строгое закрытие (Graceful Shutdown) для БД sqlite — убивать процесс нельзя, ждать `authorizationStateClosed`.
- Все `invoke()` в React идут только через слой `shared/ipc/index.ts`.
- ⚠️ ЗАВИСИМОСТИ: Никогда не удалять `@prebuilt-tdlib/darwin-arm64` из `frontend/package.json`! Хоть она и кажется мусорной/неиспользуемой со стороны фронта, из её папки Rust бакенд через `dlopen` грузит ядро Телеграма (`libtdjson.dylib`). Без неё готовое приложение падает при старте.
