# Backend (src-tauri)

Тут хранится логика на стороне системы (Rust) для Tauri-приложения.

- \`src/\` — исходники на Rust, где регистрируются команды и инициализируется Tauri (например, `main.rs`).
- \`Cargo.toml\` — управление зависимостями языка Rust.
- \`tauri.conf.json\` — ключевые настройки окна, меню и подписей для десктопа.

Для сборки и запуска в dev-окружении используется команда \`npm run tauri dev\`.
