# TG-Feed Project Status

## Текущий фокус
- **Tauri (React + Rust/Tauri):** Мелкие доработки UI, поддержка OTA-апдейтов (macOS). Разработка стабилизирована. Главная директория: `tauri/`.
- **Flutter (Flutter + TDLib):** Создание нового нативного мобильного приложения (iOS, Android, macOS), которое полностью повторит функционал `tauri`. Директория: `flutter/tgf_flutter/`. Текущий статус: Разработка архитектуры и Roadmap.

## Roadmap
- **Tauri:** Выпуск стабильной версии beta.1
- **Flutter:** Поэтапная реализация Flutter-приложения. План описан в `flutter/ROADMAP.md` (настройка TDLib, Агрегатор Ленты, UI).
