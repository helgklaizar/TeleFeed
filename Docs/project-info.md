# TG-Feed: Информация о проекте

## Суть проекта
**TG-Feed** — это альтернативный десктопный клиент Telegram, ориентированный на комфортное чтение каналов. Вместо стандартного списка чатов он агрегирует ваши подписки в виде бесконечной прокручиваемой ленты, наподобие Twitter/X или RSS-ридеров. Позволяет читать каналы и общаться в чатах в едином быстром нативном приложении.
В проекте реализованы: фильтрация по папкам Telegram, сохранение постов в "Избранное", полноэкранный просмотр медиа, интеграция с AI-агентами (OpenAI, Anthropic, Ollama, LM Studio) и темизация.

## Технологический стек и архитектура
Проект разбит на две независимые ветки:

### Beta 1 (Десктоп)
1.  **Платформа 1 (Desktop):** Web + Tauri
    *   **Frontend:** ReactJS, Vite, Zustand (state).
    *   **Backend/Native:** Rust, Tauri.
2.  **Платформа 2 (Mobile/Desktop Native):** Flutter
    *   **UI/Logic:** Flutter, Dart 3.
    *   **State:** Riverpod.

**Примечание по структуре:** Разработка разделена. Десктоп (`React+Rust`) лежит в `tauri/`. Мобайл (`Flutter`) лежит в `flutter/tgf_flutter/`. Каждая версия развивается как автономный проект, но с одинаковым продуктовым функционалом.

## Инструкции по запуску
Разработка и запуск приложения происходят в директории `tauri/`.

```bash
cd tauri
npm install
npm run tauri dev
```

При первом запуске приложение запросит ваши **Telegram API credentials** (`api_id` и `api_hash`). Их можно получить на странице: [my.telegram.org/apps](https://my.telegram.org/apps).
