# Changelog

## [Unreleased]

### Добавлено
- **Telegram Saved Messages**: Синхронизация Избранного (сохраненных сообщений) при клике на сердечко.
- **Jina AI**: Парсинг и чтение полных статей по ссылкам "Читать далее" напрямую в приложении.
- **Rust Backend**: Новая команда `get_user` для получения информации о пользователях.
- **UI**: Кнопка "Очистить кэш медиа", кнопка "Синхронизировать каналы", переключатель панели папок.
- **UX**: Анимация (вспышка) при клике на иконки глаза (просмотр) и сердца (сохранение).

### Исправлено
- **Чат**: Восстановлено отображение имен и аватарок в групповых чатах.
- **AI**: Исправлен ввод и контекст для AI Chat.

### Изменено
- **Оптимизация**: `FeedCard` обернут в `React.memo` с кастомным компаратором для предотвращения O(N) ререндеров списков при лайке/прочтении одного поста.
- **Оптимизация**: Селекторы Zustand в `FeedCard` (`useUiStore`) теперь выбирают конкретные булевы значения вместо целых коллекций (Set), что радикально снижает число холостых обновлений компонентов при скролле и кликах.
- **Оптимизация**: Исправлено предупреждение ESLint в `PostContent.jsx` путём удаления неиспользуемых параметров (`showDate`) и импортов (`useState`).

## [0.1.0-beta.1] — 2026-02-25

### Первый публичный релиз (beta)

#### Добавлено
- **Онбординг** — пошаговая настройка API credentials + авторизация
- **Лента каналов** — карточки с медиа, фильтрация по папкам Telegram
- **Группы и Личка** — Telegram-style чат-вью с bubble messages
- **Mark as read** — с анимацией slide-out
- **Избранное** — пересылка в «Стену»
- **Медиа** — фото/видео/GIF + полноэкранный просмотр
- **i18n** — поддержка EN/RU
- **Темы** — blue/green/bordeaux
- **Zoom** — Cmd+/-/0
- **ИИ-агенты** — CRUD для AI-провайдеров (OpenAI, Anthropic, Ollama и др.)
- **Storybook** — каталог UI-компонентов
- **Unit-тесты** — 37 тестов (Vitest)
- **CI** — GitHub Actions (lint + test + storybook)
- **Документация** — README, SRS, HLD, AGENTS.md

#### Безопасность
- API-ключи вводятся через UI (runtime), не хардкодятся
- CSP настроен в tauri.conf.json
- SAFETY комментарии на всех unsafe-блоках

#### Архитектура
- Tauri v2 + React 19 + Zustand 5 + React Router 7
- Один TDLib-клиент (без мульти-аккаунта)
- Code splitting через React.lazy()
- ErrorBoundary per route
- Message eviction (MAX 100/чат)
