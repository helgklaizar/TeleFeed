# SRS — Software Requirements Specification

**Проект:** TG-Feed v3  
**Версия:** 0.1.0  
**Дата:** 2026-02-25

---

## 1. Введение

### 1.1 Назначение
TG-Feed — десктопное приложение для агрегированного чтения Telegram-каналов, групп и личных чатов. Работает как альтернативный клиент с фокусом на ленту каналов.

### 1.2 Scope
- Один Telegram-аккаунт
- Только macOS (darwin-arm64)
- Только чтение + базовые действия (mark as read, forward, reply)

### 1.3 Определения
- **TDLib** — Telegram Database Library (C++ с JSON-интерфейсом)
- **Tauri** — фреймворк для десктопных приложений (Rust + WebView)
- **Стена** — специальный Telegram-канал/чат для сохранения избранных постов

## 2. Функциональные требования

### 2.1 Авторизация (FR-AUTH)
| ID | Требование | Приоритет |
|---|---|---|
| FR-AUTH-01 | Ввод номера телефона | Must |
| FR-AUTH-02 | Ввод SMS/Telegram-кода | Must |
| FR-AUTH-03 | Ввод облачного пароля (2FA) | Must |
| FR-AUTH-04 | Автоматическая авторизация при перезапуске (TDLib session) | Must |

### 2.2 Лента каналов (FR-FEED)
| ID | Требование | Приоритет |
|---|---|---|
| FR-FEED-01 | Отображение постов из подписанных каналов в хронологическом порядке | Must |
| FR-FEED-02 | Поддержка типов контента: текст, фото, видео, GIF | Must |
| FR-FEED-03 | Рендеринг entities: bold, italic, url, mention, code | Must |
| FR-FEED-04 | Фильтрация по папкам Telegram | Should |
| FR-FEED-05 | Mark as read (с анимацией) | Must |
| FR-FEED-06 | Добавление в избранное (forward в «Стену») | Should |
| FR-FEED-07 | Виртуализация списка (react-virtuoso) | Must |
| FR-FEED-08 | Полноэкранный просмотр медиа | Should |
| FR-FEED-09 | Альбомы (группировка по media_album_id) | Should |

### 2.3 Группы и Личка (FR-CHAT)
| ID | Требование | Приоритет |
|---|---|---|
| FR-CHAT-01 | Список чатов с аватаром и именем | Must |
| FR-CHAT-02 | Переход в чат → bubble-переписка | Must |
| FR-CHAT-03 | Подгрузка истории при scroll вверх | Must |
| FR-CHAT-04 | Отправка текстовых сообщений | Should |
| FR-CHAT-05 | Кнопка «назад» | Must |

### 2.4 Настройки (FR-SETTINGS)
| ID | Требование | Приоритет |
|---|---|---|
| FR-SET-01 | Профиль пользователя (аватар, имя) | Must |
| FR-SET-02 | Управление ИИ-агентами (CRUD) | Could |
| FR-SET-03 | Инструкции для ИИ | Could |
| FR-SET-04 | Список каналов (blacklist) | Should |
| FR-SET-05 | Темы (blue/green/bordeaux) | Should |
| FR-SET-06 | Выход из аккаунта | Must |

### 2.5 UI (FR-UI)
| ID | Требование | Приоритет |
|---|---|---|
| FR-UI-01 | Навигация: 4 таба (Каналы, Группы, Личка, ИИ) + Menu | Must |
| FR-UI-02 | Zoom (Cmd+/-/0) | Should |
| FR-UI-03 | Keyboard shortcuts (Cmd+1/2/3, Esc) | Should |
| FR-UI-04 | i18n (EN/RU) | Should |

## 3. Нефункциональные требования

| ID | Требование | Метрика |
|---|---|---|
| NFR-01 | Время запуска до ready | < 5 сек (при существующей сессии) |
| NFR-02 | Лента: рендеринг 500 постов | Без заметных тормозов (виртуализация) |
| NFR-03 | Память: messageStore | MAX 100 сообщений/чат |
| NFR-04 | Безопасность: API-ключи | Не в исходном коде (env!()) |
| NFR-05 | Безопасность: CSP | Настроен (не null) |
| NFR-06 | Линтинг | ESLint 0 ошибок |
| NFR-07 | Тесты | Vitest: минимум 30 unit-тестов |

---

## 4. Ограничения

- Только 1 аккаунт (без мульти-аккаунта)
- Только macOS (darwin-arm64, libtdjson.dylib)
- Нет поиска публичных каналов
- Нет отправки медиа (только текст)
- Нет голосовых/видео-звонков

---

## 5. Out of Scope (v1)

- iOS / Android / Windows
- Мульти-аккаунт
- Шифрованные чаты (Secret Chats) — поддержка TDLib есть, UI нет
- Стикеры / реакции / голосования
- Push-уведомления
